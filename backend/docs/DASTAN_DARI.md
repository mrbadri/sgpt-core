# مستندات فارسی — بک‌اند ربات بله

این سند توضیح می‌دهد اپلیکیشن بک‌اند ربات بله چگونه کار می‌کند.

---

## نمای کلی

این پروژه یک **بک‌اند** برای ربات مسنجر **بله** است که از API سازگار با تلگرام استفاده می‌کند. اپلیکیشن با **FastAPI** نوشته شده و چند ماژول اصلی دارد:

- **ربات بله** — دریافت و پاسخ به پیام‌های کاربران
- **API ربات (Bot API)** — سرویس‌های مورد استفاده از سمت ربات
- **API ادمین (Admin API)** — مدیریت، گزارش‌ها و کنترل دسترسی

---

## جریان کلی اجرا

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────┐
│   کاربر بله     │────▶│  ربات بله        │────▶│  بک‌اند     │
│  (پیام/کامند)   │     │  (Polling)       │     │  FastAPI    │
└─────────────────┘     └──────────────────┘     └──────┬──────┘
                                                        │
                                                        ▼
                                                ┌──────────────┐
                                                │  PostgreSQL  │
                                                └──────────────┘
```

۱. کاربر پیام یا کامند در بله ارسال می‌کند.  
۲. ربات با **Polling** آن را از سرور بله می‌گیرد.  
۳. بک‌اند پردازش می‌کند و در صورت لزوم در دیتابیس ذخیره می‌کند.  
۴. پاسخ از طریق ربات به کاربر برگردانده می‌شود.

## سرویس ربات بله (Bot Service)

### راه‌اندازی

- اگر `BALE_BOT_TOKEN` در محیط تنظیم شده باشد، هنگام **startup** اپ، سرویس ربات به‌صورت خودکار:
  1. اتصال به API بله را تست می‌کند  
  2. **Polling** را در یک thread جداگانه شروع می‌کند  
  3. منتظر پیام‌های کاربران می‌ماند  

- Polling در یک **thread جدا** اجرا می‌شود تا FastAPI مسدود نشود.

### کامندهای ربات

| کامند | کار |
|-------|-----|
| `/start` | پیام خوش‌آمدگویی و ذخیره کاربر در دیتابیس |
| `/echo <متن>` | برگرداندن همان متن |

### پردازش `/start`

۱. استخراج `payload` در صورت وجود (مثلاً از لینک دعوت)  
۲. ذخیره کاربر در جدول `bot_users` (اگر قبلاً وجود نداشته باشد)  
۳. ارسال پیام خوش‌آمدگویی به زبان فارسی

### مدل کاربر ربات

هر کاربری که `/start` بزند، در جدول `bot_users` با فیلدهای زیر ذخیره می‌شود:

- `bale_user_id` — شناسه کاربر در بله  
- `username` — نام کاربری  
- `first_name` / `last_name` — نام و نام‌خانوادگی  
- `is_bot` — ربات بودن یا نه  
- `language_code` — زبان  
- `created_at` / `updated_at` — زمان ایجاد و به‌روزرسانی

## APIها

### Bot API (`/api/bot`)

| مسیر | توضیح |
|------|-------|
| `/invites` | پذیرش دعوت، دریافت جزئیات دعوت |
| `/phone` | تأیید شماره موبایل |
| `/contests` | مسابقات |
| `/training` | آموزش |
| `/health` | سلامت سرویس |

بیشتر این مسیرها در حال حاضر stub هستند و پیاده‌سازی کامل ندارند.

### Admin API (`/api/admin`)

| مسیر | توضیح |
|------|-------|
| `/auth` | احراز هویت ادمین |
| `/groups` | مدیریت گروه‌ها |
| `/leaders` | رهبران |
| `/members` | اعضا |
| `/invites` | دعوت‌نامه‌ها |
| `/contests` | مسابقات |
| `/exports` | خروجی داده |
| `/audit` | لاگ امنیتی |
| `/users` | کاربران |

احراز هویت ادمین هنوز پیاده‌سازی کامل نشده است.

### نقاط مشترک

| مسیر | توضیح |
|------|-------|
| `GET /` | اطلاعات اپ و نسخه |
| `GET /health` | سلامت کلی |
| `GET /bot/status` | وضعیت ربات (در حال اجرا یا خاموش) |
| `GET /docs` | مستندات Swagger OpenAPI |

## دیتابیس

- **DBMS:** PostgreSQL  
- **ORM:** SQLAlchemy 2  
- **مایگریشن:** Alembic  

جلسه دیتابیس از طریق `get_db_session()` ساخته می‌شود و در dependencyهای FastAPI استفاده می‌شود.

## تنظیمات (متغیرهای محیطی)

| متغیر | توضیح | پیش‌فرض |
|-------|-------|---------|
| `DATABASE_URL` | آدرس اتصال PostgreSQL | `postgresql://user:password@localhost:5432/bale_bot` |
| `BALE_BOT_TOKEN` | توکن ربات بله | — (الزامی برای ربات) |
| `BALE_API_URL` | آدرس API بله | `https://tapi.bale.ai/bot{0}/{1}` |
| `ADMIN_SECRET_KEY` | کلید امنیتی ادمین | — |
| `SECRET_KEY` | کلید عمومی برنامه | `change-me-in-production` |
| `DEBUG` | حالت دیباگ | `false` |
| `LOG_LEVEL` | سطح لاگ | `INFO` |

## حالت‌های مختلف اجرا

### اجرای مستقیم

```bash
export PYTHONPATH=$(pwd)/src:$PYTHONPATH
uvicorn src.app.main:app --reload --host 0.0.0.0 --port 8000
```

### اجرا با Docker Compose

```bash
docker-compose -f docker/docker-compose.dev.yml up
```

### مایگریشن دیتابیس

```bash
export PYTHONPATH=$(pwd)/src:$PYTHONPATH
alembic upgrade head
```
